Lesson 1

HTML= Hypertext Markup Language
Made up of text content (what you see), markup (what it looks like), references to other documents (images and videos), and links to other pages.

An element is made up of three parts:
Opening tag
Contents
Closing tag*

**Note that a closing tag is not needed if there is no content, such as breaks

Attributes assign value to an element's content
For example, the attribute name href is used to create links.
< a href = www.reddit.com> creates a link to www.reddit.com
